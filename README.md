### 🚀 Featured Projects

#### 🐮 Self-Supervised Learning for Behavioral Analysis
*Ongoing "Fil Rouge" Project*
* **Goal:** Classification of cow behaviors (e.g., rumination) from video surveillance (RGB & Infrared) without extensive annotation.
* **Methodology:**
    * **Detection:** YOLO for bounding box detection.
    * **Segmentation:** Utilizing **SAM 3 (Segment Anything Model)** for robust segmentation, handling occlusions better than previous iterations.
    * **Classification:** Implementing **VideoMAE** (Masked Autoencoders) for self-supervised learning on video data.
* **Robustness:** evaluating model performance under data scarcity (25%, 50%, 80% datasets) and occlusion scenarios.
* **Tools:** PyTorch, Hugging Face, SAM 3.

#### 🧬 Reproducible RNA-seq Analysis Pipeline
*Bioinformatics Project*
* **Goal:** Reproduction of a genomic analysis study from raw sequencing data to biological insights.
* **Methodology:**
    * Built a fully reproducible pipeline using **Nextflow** and **Docker** containers.
    * Performed Quality Control, Mapping, and Quantification.
    * Conducted statistical analysis in **R** to identify differentially expressed genes (focusing on translation-related genes).
* **Tools:** Nextflow, Docker, R (DESeq2, ggplot2).
